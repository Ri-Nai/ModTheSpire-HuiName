# 以辉之名 (HuiName)

这是一个《杀戮尖塔》模组，旨在将游戏中的特定文本替换为“辉”字或相关词汇，以增加趣味性。

## 功能介绍

1.  **同音字替换**：自动将发音为 `hui` 和 `fei` 的常见汉字替换为“辉”。
    *   例如：“会”、“回”、“飞”、“非” 等都会变成 “辉”。
2.  **特定词汇替换**：内置了一些有趣的特定词汇替换规则。
    *   化学物X -> 辉药
    *   冻结之眼 / 天眼 -> 辉眼
    *   旋风斩 -> 辉风斩
    *   死亡收割 -> 死亡辉哥
    *   编译冲击 -> 辉译冲击

## 自定义配置

本模组支持高度自定义。

1.  **配置文件**：模组首次运行后，会在游戏根目录下生成 `HuiNameConfig.json` 文件。
2.  **修改规则**：您可以编辑该 JSON 文件，添加或修改替换规则。
    *   `target`: 替换后的目标词（例如 "辉"）。
    *   `sources`: 需要被替换的源词列表（例如 ["会", "回", ...]）。
3.  **重载配置**：修改配置文件后，无需重启游戏。进入 **Mods** 菜单，找到 **以辉之名**，点击 **Config** 按钮，然后点击 **重载配置** 即可立即生效。

## 安装方法

1.  确保已安装 **ModTheSpire** 和 **BaseMod**。
2.  将本模组的 jar 文件放入游戏的 `mods` 文件夹中。
3.  启动 ModTheSpire 并勾选本模组。
