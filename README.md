# 以辉之名 (HuiName)

这是一个《杀戮尖塔》模组，旨在将游戏中的特定文本替换为“辉”字或相关词汇，以增加趣味性。

## 功能介绍

1.  **同音字替换**：自动将发音为 `hui` 和 `fei` 的常见汉字替换为“辉”。
    *   例如：“会”、“回”、“飞”、“非” 等都会变成 “辉”。
2.  **特定词汇替换**：内置了一些有趣的特定词汇替换规则。
    *   化学物X -> 辉药
    *   冻结之眼 / 天眼 -> 辉眼
    *   旋风斩 -> 辉风斩
    *   死亡收割 -> 死亡辉哥
    *   编译冲击 -> 辉译冲击

## 自定义配置

本模组支持高度自定义，您可以通过游戏内的配置菜单轻松管理替换规则。

1.  **配置菜单**：进入 **Mods** 菜单，找到 **以辉之名**，点击 **Config** 按钮即可打开配置界面。
2.  **添加规则**：
    *   在界面中输入 **目标词**（例如 "辉"）和 **源词**（例如 "会,回"）。
    *   点击 **添加** 按钮保存规则。
    *   *注意：添加新规则后需要重启游戏才能生效。*
3.  **管理规则**：
    *   配置界面下方列出了当前生效的所有规则。
    *   您可以点击规则右侧的 **[删除]** 按钮来移除不需要的规则（删除后同样需要重启游戏）。
4.  **配置文件位置**：
    *   配置数据存储在 `preferences/HuiNameConfig.properties` 中。
    *   您也可以点击配置界面上的 **打开配置文件** 按钮直接查看或备份配置文件。

## 安装方法

1.  确保已安装 **ModTheSpire** 和 **BaseMod**。
2.  将本模组的 jar 文件放入游戏的 `mods` 文件夹中。
3.  启动 ModTheSpire 并勾选本模组。
